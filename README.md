# DSiSaveManager
A save manager for your Nintendo DSi save data

## How to run the script
Run the following commands:
```shell
sudo apt install git
git clone https://github.com/MKstarFromSwitch/DSiSaveManager.git
cd DSiSaveManager
```

Open config.ini and set SdRoot from "/path/to/sd" to the path of your SD card.

Then run it by typing ```./DSiSaveManager.sh```
